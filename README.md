# Daily Task Tracker 📋

A simple, clean, and responsive web application to help you track your daily tasks.

## Features

✨ **Core Features:**
- ✅ Add new tasks easily
- ✅ Mark tasks as completed
- ✅ Delete individual tasks
- ✅ Filter tasks (All, Active, Completed)
- ✅ Clear all completed tasks at once
- ✅ View statistics (Total, Completed, Remaining tasks)
- 💾 Local storage persistence - your tasks are saved automatically
- 📱 Fully responsive design for mobile, tablet, and desktop

## How to Use

1. **Open the website:** Open `index.html` in your web browser
2. **Add a task:** Type your task in the input field and click "Add Task" or press Enter
3. **Complete a task:** Check the checkbox next to a task to mark it as done
4. **Delete a task:** Click the "Delete" button to remove a task
5. **Filter tasks:** Use the filter buttons to view All, Active, or Completed tasks only
6. **Clear completed:** Click "Clear Completed Tasks" to remove all finished tasks at once

## Data Storage

Your tasks are automatically saved to your browser's LocalStorage. This means:
- ✅ Tasks persist even after closing the browser
- ✅ Each browser has its own task list
- ✅ Clearing browser data will remove your tasks
- ⚠️ Private browsing mode won't save data between sessions
